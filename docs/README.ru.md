## 🚀 Быстрый старт

### Требования

| Зависимость | Версия |
|-------------|--------|
| macOS | 26.0+ (или выше) |
| Xcode | 26.0+ |
| Homebrew | Последняя версия |

### Установка зависимостей

```bash
brew install go
```

> 📝 **Примечание**: libusb-1.0 теперь включен в приложение, ручная установка не требуется.

### Сборка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/wang93wei/SwiftMTP.git
cd SwiftMTP

# Сборка моста Go
./Scripts/build_kalam.sh

# Открытие в Xcode и запуск
open SwiftMTP.xcodeproj
```

> 📝 **Примечание**: Файлы конфигурации проекта были добавлены в систему контроля версий. После клонирования просто откройте проект в Xcode и начните сборку без дополнительной настройки.

> 💡 **Совет**: После подключения устройства Android выберите на устройстве режим **Передача файлов (MTP)**, чтобы начать использовать.

### Создание установочного пакета

```bash
# Упаковка упрощенная (без сертификата разработчика)
./Scripts/create_dmg_simple.sh

# Полная упаковка (требует сертификат разработчика)
./Scripts/create_dmg.sh
```

DMG-файл будет создан в каталоге `build/`.

## 📖 Руководство пользователя

### Подключение устройства

1. Подключите устройство Android к Mac через USB
2. Выберите на устройстве режим **Передача файлов (MTP)**
3. SwiftMTP автоматически обнаружит и отобразит устройство

### Операции с файлами

| Операция | Метод |
|----------|-------|
| Просмотр файлов | Двойной клик по папкам для входа, хлебные крошки для возврата |
| Скачивание файла | Правый клик по файлу → **Скачать** |
| Пакетное скачивание | Множественный выбор файлов → Правый клик → **Скачать выбранные файлы** |
| Загрузка файла | Кнопка **Загрузить файл** на панели инструментов или перетаскивание файлов в окно |
| Загрузка перетаскиванием | Перетащите файлы в окно браузера файлов для загрузки |

### Настройки языка

1. Откройте окно **Настройки** (⌘ + ,)
2. Выберите язык на вкладке **Общие**
3. Доступные языки:
   - **По умолчанию системы** - Следует языку macOS
   - **English** - Английский интерфейс
   - **中文** - Интерфейс на упрощенном китайском
   - **日本語** - Интерфейс на японском
   - **한국어** - Интерфейс на корейском
   - **Русский** - Интерфейс на русском
   - **Français** - Интерфейс на французском
   - **Deutsch** - Интерфейс на немецком
4. Интерфейс приложения обновит язык немедленно
5. **Меню и выбор файлов** требуют перезапуска приложения для вступления в силу, система предложит немедленный перезапуск

### Технологический стек

- **Языки**: Swift 6+, Go 1.23+
- **UI-фреймворк**: SwiftUI
- **Библиотека MTP**: go-mtpx (на основе libusb-1.0)
- **Архитектурный шаблон**: MVVM
- **Метод моста**: CGO
- **Интернационализация**: Фреймворк локализации Swift (NSLocalizedString)

## ⚠️ Известные ограничения

1. Песочница должна быть отключена для доступа к USB-устройствам
2. Скорость передачи ограничена протоколом MTP
3. В настоящее время поддерживается только загрузка одиночного файла (загрузка папок не поддерживается)
4. Реализация Liquid Glass UI неполна и содержит ошибки, которые необходимо исправить
5. Мы приветствуем больше участников для улучшения кодовой базы

## 🔧 Устранение неполадок

### Устройство не обнаруживается

```
✓ Убедитесь, что устройство находится в режиме MTP
✓ Попробуйте отключить и снова подключить USB-кабель
✓ Перезапустите приложение
✓ Проверьте, поддерживает ли USB-кабель передачу данных
```

### Ошибки сборки

```bash
# Пересборка моста Go
./Scripts/build_kalam.sh

# Очистка и пересборка
xcodebuild clean
xcodebuild
```
